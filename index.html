Sistema de vendas Web

Desenvolvi um site de e-commerce com foco na experiência do usuário, onde é possível visualizar produtos, adicioná-los ao carrinho e simular uma compra de forma organizada. O sistema gerencia os itens no carrinho, atualizando quantidades e valores em tempo real, deixando a estrutura pronta para futura integração com back-end e métodos de pagamento. O projeto foi desenvolvido utilizando HTML, CSS e Js, com ênfase em responsividade, interatividade e boas práticas de desenvolvimento front-end.
